PRE-REQUIS:
    - Installer python avec les librairies à utiliser,
    - on a utilisé la bibliothèque 'Curses' pour la saisie et l’affichage.

POUR EXECUTER:
    - Cliquer deux fois sur le fichier tetris.py ou exécuter cet commande sur votre console " python tetris.py ".

COMMANDES:
    - 'Gauche' : pour aller vers la gauche,
    - 'Droite' : pour aller vers la droite,
    - 'Bas'    : pour accélérer la descente,
    - 'X'      : pour tourner vers le sens des aiguilles d'une montre,
    
FONCTIONNALITES SUPPLEMENTAIRES:
    - 'Espace' : pour la chute dure,
    - 'Z'      : pour tourner vers le sens contraire des aiguilles d'une montre,
    - 'C'      : pour placer dans la corbeille,
    - 'R'      : pour restarter le jeu.
